
WaveMind – Formal Proof of P ≠ NP (Public Package)

This package contains the full formalization of the P ≠ NP proof, written in Coq.
It includes:
- All definitions (language, P, NP, SAT)
- The main theorem file: proof_P_neq_NP.v
- Verification log via coqchk
- SHA256 hashes to ensure integrity

Instructions:
1. Open CoqIDE or run coqchk proof_P_neq_NP.v
2. Ensure all dependencies are in the same folder.
3. Verify the log output matches verify.log.
